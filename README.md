# Kenya Demographics Units
An updated data of Kenya administrative and political units

## Units Available
* Administrative Units
  * Counties (under-dev)
  * Sub-counties (under-dev)
  * Districts (under-dev)
  * Locations (under-dev)
  * Sub-locations (under-dev)
* Political Units
  * Counties (under-dev)
  * Constituencies (under-dev)
  * Wards (under-dev)
